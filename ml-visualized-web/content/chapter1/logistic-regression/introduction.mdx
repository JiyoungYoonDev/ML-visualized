<Takeaway>
  <TakeawayItem value='concept-mastery' title='Concept Mastery'>
    <div className='space-y-3'>
      <p className="text-sm sm:text-base">
        Logistic regression definition and how it differs from a plain linear regression:
      </p>
      <ul className='list-disc pl-5 space-y-2 text-sm sm:text-base'>
        <li>Logistic regression is a classification model that predicts probabilities (ranging from 0 to 1) based on input x.</li>
        <li>Linear regression predicts a continuous numerical value (e.g., house prices).</li>
        <li>Logistic regression predicts the likelihood of an event (e.g., rain, spam, disease).</li>
        <li>Key advantage: It provides a <strong>confidence level</strong> for the prediction.</li>
      </ul>
    </div>
  </TakeawayItem>

<TakeawayItem value='why-logistic' title='Why Logistic?'>
  <div className='space-y-3 text-sm sm:text-base'>
    <p>
      The main reason is that it outputs probabilities between 0 and 1. Linear
      regression can output any real number (e.g., -0.5 or 1.7), which is
      meaningless for "likelihood."
    </p>
    <p className='mt-2 italic text-muted-foreground'>
      Valid probabilities must lie in [0,1]. Logistic regression ensures this
      "squashing" effect.
    </p>
  </div>
</TakeawayItem>

</TakeawayItem>

  <TakeawayItem value='optimization' title='Optimization'>
    <p className="text-sm sm:text-base">
      Understand <strong>margin intuition</strong> and how to minimize loss using <strong>gradient descent</strong> updates.
    </p>

</Takeaway>
